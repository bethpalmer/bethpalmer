Recreation of a Classic Arcade Game

Player control:

How to play:

Use the arrow keys on your keyboard to direct the character safely to the waters edge but mind out for those enemy bugs on the way!

How the game was created

This game was created using object-orientated JavaScript prgramming. The game engine and graphics were provided to me and I added the functionality of:

  • the enemy bugs travelling horizontally across the screen
  • the ability of the character to move within the game bounds listening for events from the keyboard arrow keys
  • the detection of collisions between the enemy bugs and the character to end the game
  • the success of the character reaching the waters edge to win the game
